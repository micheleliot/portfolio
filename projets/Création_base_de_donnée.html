Création d'une base de données

Ce projet s'effectuait en binôme et consistait à créer une base de données à partir de données venant du sauvetage du Titanic, Donc j'ai dû créer plusieurs tables de données que ce soit les barques de sauvetages et leurs emplacements sur le bateaux, les différents passagers et le positionnement de leur cabine ou encore les passagers qui sont morts ou ceux qui ont été sauvés et leur temps de sauvetage.

Mon rôle dans ce projet

Dans ce projet, j'ai créé des table contenant diverse information sur le titanic ainsi qu'une partit du compt rendu.

Les compétences utilisées et apprises

Durant ce projet, j'ai utilisé mes compétences avec PostgreSQL pour créer une base de donnée mais j'ai beaucoup appris car je n'avais jamais créé autant de table donc la cohérence des données était primordiale;
