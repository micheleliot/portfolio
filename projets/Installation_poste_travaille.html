Installation d'un poste de travail

Ce projet consistait à mettre en place le système d'exploitation DEBIAN sur une machine virtuelle de sorte que le poste de travail puisse être opérationel, ce qui nécessitait la configuration du système d'exploitation et l'installation de packages tels que git, java, intellij.

Mon rôle dans ce projet

Dans ce projet, j'ai installer DEBIAN sur une machine virtuelle puis installer le logiciel git pour enregistrer au fur et à mesure nos fichier ensuite j'ai installer le package java pour pouvoir coder en java et enfin intellij pour avoir un IDE pour java

Les compétences utilisées et apprises

Durant ce projet, j'ai utilisé mes compétences avec le système de commande (BASH) pour installer tout les packages nécesaire mais j'ai beaucoup appris sur les façons de créer un système debian sur une machine virtuelle.
