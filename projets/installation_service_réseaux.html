Installation_service_réseaux

Dans ce projet, j'ai installé le système d'exploitation DEBIAN sur une machine virtuelle sans interface graphique, sur laquelle j'ai installé une liaison SSH pour accéder au serveur sur le réseaux et PostgreSQL pour pouvoir gérer des bases de données , un serveur Apache disponible sur réseau local où on peut accéder à un réseaux PHP et un réseau PhpPgAdmin.

Mon rôle dans ce projet

Dans ce projet, j'ai installer DEBIAN sur une machine virtuelle puis installer PostgreSQL ainsi qu'un serveur Apache ainsi qu'une page PhpPgAdmin sur le serveur apache directement connecté au base de donnée crée sur la machine virtuell Puis j'ai rédigé un guide complet en anglais de comment procéder pour installer un service de serveur.

Les compétences utilisées et apprises

Durant ce projet, j'ai utilisé mes compétences avec PostgreSQL pour l'installer et créer une base de donnée ainsi qu'avec le système de commande (BASH) mais j'ai beaucoup appris sur les façons de créer une liaison externe tel qu'une page PhpPgAdmin pouvant intéragir avec les bases présent sur la machine virtuelle pouvant être accessible depuis un autre ordinateur du réseaux local.
